How a site uses HTML and CSS properly.

Introduction

How HTML works

HTML is the main part of the site. The tags that display the site normally are used here. These can be arranged into either paragraphs, or lists.

How CSS works

CSS determines how the site appears. Using CSS is very useful, but remember that using CSS is slightly more complex than usual. you need selectors, classes, and IDs.

HTML fundamentals

Tags

Tags are the main thing that are used in HTML. These are used to write text, make lists, add images and videos, and make links that go to other pages.

Elements

Each tag is used to make various elements, and are important when making a website!

Attributes

    Attributes are used in elements and tags, allowing you to adjust those, and examples include things like:
  • Headers, because you can use them for titles of pages, or paragraphs.
  • Lists, so you can order them, when adding text or links
  • Images and Videos, because you can adjust the size of them.

    • still not putting them within distance of other tags, but that's what CSS is for!

    CSS fundamentals

    Selectors

    Selectors are used for ways to add in adjustments on text. they can be used on tags, classes, and IDs (more on the other 2 below)

    Classes

    Classes can be used the same way, but the main difference is that using a class ends up affecting EVERYTHING in the class, via a period and the tag after unless you put a CSS to something directly in the site.

    IDs

    No, IDs are not your personal information! Instead, IDs can affect a certain thing in the code, and is usually used with a #.
